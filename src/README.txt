# Jogo da Forca - SDTP1

## Requisitos
- Java 8 ou superior

Na pasta raiz:
## Compilação
javac src/*.java

## Execução do servidor
java -cp src Server

## Execução do cliente
java -cp src Client

## Notas importantes
- O ficheiro words.txt deve estar na pasta raiz do projeto
- O programa deve ser executado a partir da pasta raiz

## Ou executar normalmente no IDE
